\doxysubsection{Detailed Description}
Attribute that specifies a class as an editor for an extended \doxylink{namespace_timeline}{Timeline} type. 

Use this attribute on a class that extends Clip\+Editor, Track\+Editor, or Marker\+Editor to specify either the Playable\+Asset, Marker, or Track\+Asset derived classes for associated customization.
