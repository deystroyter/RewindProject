 The {\bfseries{Single}} view displays information about a single set of profiling data. You can use it to analyze how markers perform across frames. The view is divided into several frames, which contain information on frame timings, as well as min, max, median, mean, and lower / upper quartile values for frames, threads, and markers.

For information on navigating the window, see the \href{profile-analyzer-window.html/\#window-navigation}{\texttt{ Profile Analyzer window navigation}} documentation.

To load data into the {\bfseries{Single}} view, select the {\bfseries{Pull Data}} button in the frame control pane, and the Profile Analyzer pulls in any data in the \href{https://docs.unity3d.com/Manual/Profiler.html}{\texttt{ Profiler}} window. Alternatively, select the {\bfseries{Load}} button to load Profile Analyzer (.pdata) data you have saved from a previous session.

\texorpdfstring{$>$}{>}\mbox{[}!\+NOTE\mbox{]} \texorpdfstring{$>$}{>}If you select the {\bfseries{Load}} option, the data must be in the Profile Analyzer .pdata format. If you have data from the Profiler in the .data file format, open it in the Profiler first, and then select the {\bfseries{Pull Data}} button in the Profile Analyzer.

For more information on how to pull data into the Profile Analyzer, see the workflow documentation on \doxysectlink{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4}{Collecting and viewing data}{0}.\hypertarget{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation_0i_2single-view_autotoc_md431}{}\doxysection{\texorpdfstring{Marker details list}{Marker details list}}\label{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation_0i_2single-view_autotoc_md431}
The {\bfseries{Marker details}} pane contains a sortable list of markers with a number of useful statistics. If you select a marker in the list, the {\bfseries{Marker Summary}} panel displays in depth information on the marker. Each marker in the list is an aggregation of all the instances of that marker, across all filtered threads and in all ranged frames.

You can filter the columns in the {\bfseries{Marker details}} to a more relevant set. This is particularly useful if you want to filter out irrelevant data when you look for {\bfseries{Time}} or {\bfseries{Count}} values. To filter the columns, select the {\bfseries{Marker columns}} dropdown from the {\bfseries{Filters}} pane. For more information on how to filter data, see the \doxysectlink{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation_0i_2filtering-system}{Filters}{0} documentation.\hypertarget{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation_0i_2single-view_autotoc_md432}{}\doxysubsection{\texorpdfstring{Marker details columns and groups}{Marker details columns and groups}}\label{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation_0i_2single-view_autotoc_md432}
By default, the {\bfseries{Marker columns}} dropdown in the {\bfseries{Filters}} pane has six preset column layouts that you can use to adjust the layout of the {\bfseries{Marker details}} pane. They are\+:


\begin{DoxyItemize}
\item {\bfseries{Time and count\+:}} Displays information on the average timings and number of times the markers were called.
\item {\bfseries{Time\+:}} Displays information on the average timings of the markers.
\item {\bfseries{Totals\+:}} Displays information about the total amount of time the markers took on the whole data set.
\item {\bfseries{Time with totals\+:}} Displays information about both the average and total times of the markers.
\item {\bfseries{Count totals\+:}} Displays information about the total number of times the markers were called.
\item {\bfseries{Count per frame\+:}} Displays information on the average total per frame the markers were called.
\item {\bfseries{Depths\+:}} Displays information on where the markers are in the Hierarchy. For more information, see the documentation on Depth Slices in \href{filtering-system.html\#depth-slice}{\texttt{ Filters pane}}.
\item {\bfseries{Threads\+:}} Displays the name of the thread that the markers appear on. For more information, see the documentation on the Thread window in \href{filtering-system.html\#thread-window}{\texttt{ Filters pane}}.
\end{DoxyItemize}

You can also use the {\bfseries{Custom}} column layout, to select your own custom mix of columns to add to the layout. To do this, right-\/click on the header of any column, and manually enable or disable any of the columns as necessary.
