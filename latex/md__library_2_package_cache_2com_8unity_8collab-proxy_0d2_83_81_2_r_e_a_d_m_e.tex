This package provides an in-\/editor interface for teams to work with \href{https://unity.com/solutions/version-control}{\texttt{ Unity Version Control}} (\doxylink{namespace_unity}{Unity} VCS), our leading version control solution, directly in \doxylink{namespace_unity}{Unity}.

{\itshape Note this project is the natural evolution of the old Collaborate package, hence its name.}

\href{https://docs.unity.com/ugs/en-us/manual/devops/manual/vcs-plugins/unityeditor-plugin/unity-version-control-package}{\texttt{ Documentation}} -\/ \href{https://docs.unity3d.com/Packages/com.unity.collab-proxy@2.1/changelog/CHANGELOG.html}{\texttt{ Changelog}} -\/ \href{https://unity-ci.cds.internal.unity3d.com/project/259}{\texttt{ Yamato}}\hypertarget{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_r_e_a_d_m_e_autotoc_md158}{}\doxysection{\texorpdfstring{Compatibility}{Compatibility}}\label{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_r_e_a_d_m_e_autotoc_md158}
The minimum supported version of the \doxylink{namespace_unity}{Unity} Editor is 2021.\+3 LTS.

Windows and mac\+OS are officially supported.

The solution is exclusively targeting .Net\+Standard 2.\+0, and will not work with the legacy \doxylink{namespace_mono}{Mono} runtime.\hypertarget{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_r_e_a_d_m_e_autotoc_md159}{}\doxysection{\texorpdfstring{Maintenance}{Maintenance}}\label{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_r_e_a_d_m_e_autotoc_md159}
This project is currently maintained by the VCS Ecosystem team (@vcs-\/ecosystem-\/team), part of UGS Dev\+Ops.
