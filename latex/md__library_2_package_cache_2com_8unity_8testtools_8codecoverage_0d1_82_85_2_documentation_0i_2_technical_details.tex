The package is a client of the coverage API. For more information, see the \href{https://docs.unity3d.com/ScriptReference/TestTools.Coverage.html}{\texttt{ coverage API\textquotesingle{}s documentation}}. The package uses a combination of this API and C\# reflection to output the test coverage data in the \doxylink{namespace_open_cover}{Open\+Cover} format. Optionally, a third-\/party report generator will then parse the \doxylink{namespace_open_cover}{Open\+Cover} data and generate a report (HTML, Sonar\+Qube, Cobertura, LCOV).\hypertarget{md__library_2_package_cache_2com_8unity_8testtools_8codecoverage_0d1_82_85_2_documentation_0i_2_technical_details_autotoc_md866}{}\doxysection{\texorpdfstring{Requirements}{Requirements}}\label{md__library_2_package_cache_2com_8unity_8testtools_8codecoverage_0d1_82_85_2_documentation_0i_2_technical_details_autotoc_md866}

Code Coverage includes the following known limitations\+:


\begin{DoxyItemize}
\item Code Coverage currently only supports the \href{https://github.com/OpenCover/opencover}{\texttt{ Open\+Cover}} format.
\item Code Coverage currently only supports code run in the Editor and not in Standalone/\+Player.
\item NPath Complexity calculation and Branch Coverage are not implemented at present so they will always appear as zero in the coverage report. 
\end{DoxyItemize}
