\begin{DoxyCompactList}\small\item\em A test fails if \doxylink{namespace_unity}{Unity} logs a message other than a regular log or warning message. Use {\ttfamily Log\+Assert} to check for an expected message in the log so that the test does not fail when \doxylink{namespace_unity}{Unity} logs the message. Use {\ttfamily Log\+Assert.\+Expect} before running the code under test, as the check for expected logs runs at the end of each frame.
