 Summary of changes in Profile Analyzer version 1.\+2.\+2.

The main updates in this release include\+:\hypertarget{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation_0i_2whats-new_autotoc_md448}{}\doxysection{\texorpdfstring{Added}{Added}}\label{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation_0i_2whats-new_autotoc_md448}

\begin{DoxyItemize}
\item Added support for removing vsync time, so we can see actual CPU duration over multiple frames. A drop down has been added to \textquotesingle{}remove\textquotesingle{} a marker from the analysis and has entries for "{}\+FPS Wait"{}, "{}\+Present Wait"{} and "{}\+Custom"{} where you can select any marker from the table using the right click context menu to "{}\+Remove Marker"{}.
\item Added optional standard deviation (SD) column into marker table (on single view) for sorting data based on variation of marker timings in over the frames.
\item Added export of the comparison table as CSV.
\end{DoxyItemize}
