 You can use a Custom Event Trigger node to trigger a matching Custom Event node in your application. For more information on custom Events, see \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-custom-events}{Custom Events}{0}.

\begin{quote}
\mbox{[}!\+NOTE\mbox{]} Before you add a Custom Event Trigger node, you must add and configure a Custom Event node in a Script Graph. For more information, see \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-add-custom-event-node}{Add a Custom Event node}{0}. \end{quote}
To add a Custom Event Trigger node to a Script Graph\+:


\begin{DoxyEnumerate}
\item \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-open-graph-edit}{Open the Script Graph}{0} where you want to add a Custom Event Trigger node. This can be the same graph or a different graph from where you added a Custom Event node.
\item \mbox{[}!include\doxylink{vs-open-fuzzy-finder_8md}{open-\/fuzzy-\/finder}\mbox{]}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Go to {\bfseries{Events}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select the {\bfseries{Custom Event Trigger}} node to add it to the graph.
\item In the {\bfseries{Name}} input port\textquotesingle{}s field, enter the name of the Custom Event node you want to trigger, exactly as it appears on the Custom Event node, through one of the following methods\+:
\begin{DoxyItemize}
\item Enter the name in the field next to the {\bfseries{Name}} input port.
\item Attach a node that outputs the name as a string value to the {\bfseries{Name}} input port.
\end{DoxyItemize}
\item In the {\bfseries{Game\+Object}} field, indicated by the Game\+Object icon on the node, specify the Game\+Object that you want to trigger your Event. This doesn\textquotesingle{}t have to be the same Game\+Object as the Custom Event node. Do one of the following\+:
\begin{DoxyItemize}
\item Select the object picker (circle icon) and select the Game\+Object.
\item Attach a node to the field\textquotesingle{}s data input port that outputs the Game\+Object.
\item Leave the field as the default value of {\bfseries{This}} to use the Game\+Object from your currently open Script Graph\textquotesingle{}s Script Machine.
\end{DoxyItemize}
\item In the {\bfseries{Arguments}} field, enter the number from your Custom Event node\textquotesingle{}s {\bfseries{Arguments}} field.
\end{DoxyEnumerate}

\begin{quote}
\mbox{[}!\+NOTE\mbox{]} All arguments on a Custom Event Trigger node must receive input values, even if the Custom Event node doesn\textquotesingle{}t use those arguments. Otherwise, Visual Scripting displays an error in the Graph Inspector for the Custom Event Trigger node. \end{quote}
