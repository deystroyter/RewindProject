 \doxylink{namespace_unity}{Unity} UI is a UI toolkit for developing user interfaces for games and applications. It is a Game\+Object-\/based UI system that uses Components and the Game View to arrange, position, and style user interfaces.

You cannot use \doxylink{namespace_unity}{Unity} UI to create or change user interfaces within the \doxylink{namespace_unity}{Unity} Editor.\hypertarget{md__library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2ugui_autotoc_md1459}{}\doxysection{\texorpdfstring{Installing Unity UI}{Installing Unity UI}}\label{md__library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2ugui_autotoc_md1459}
\doxylink{namespace_unity}{Unity} UI is a core package. A version of it is included in each \doxylink{namespace_unity}{Unity} release.

To remove this package, or reinstall it after removal, follow the instructions in the \href{https://docs.unity3d.com/Manual/upm-ui-actions.html}{\texttt{ Package Manager documentation}}.\hypertarget{md__library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2ugui_autotoc_md1460}{}\doxysection{\texorpdfstring{Getting documentation}{Getting documentation}}\label{md__library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2ugui_autotoc_md1460}
