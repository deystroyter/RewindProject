 To avoid any problems with automatically generated files, exclude some Visual Scripting files from your version control solution.

To exclude files from version control, include a file or configure your settings to specify which files and folders to exclude\+:


\begin{DoxyEnumerate}
\item Create a new file at the root of your project directory.

\texorpdfstring{$>$}{>}\mbox{[}!\+TIP\mbox{]} \begin{quote}
The root of your project directory is at the level above your {\bfseries{Assets}} folder. \end{quote}

\item Name the file based on your chosen version control system\+:
\begin{DoxyItemize}
\item {\bfseries{Git}}\+: {\ttfamily .gitignore}. For more information, see \href{https://git-scm.com/docs/gitignore}{\texttt{ Git\textquotesingle{}s documentation on gitignore}}.
\item {\bfseries{\doxylink{namespace_unity}{Unity} Collab}}\+: {\ttfamily .collabignore}. For more information, see the \href{https://docs.unity3d.com/Manual/UnityCollaborateIgnoreFiles.html}{\texttt{ Unity User Manual}}.
\item {\bfseries{Subversion}}\+: Ignore the files from your {\ttfamily svn\+:ignore} property or runtime configuration options. For more information, see Subversion\textquotesingle{}s documentation on \href{https://svnbook.red-bean.com/en/1.7/svn.advanced.props.special.ignore.html}{\texttt{ Ignoring Unversioned Items}}.
\end{DoxyItemize}
\item Open the file in a text editor.
\item Add the appropriate files or file patterns to your ignore file or configuration. For an example and more information, see Ignore file template.
\end{DoxyEnumerate}

\begin{quote}
\mbox{[}!\+NOTE\mbox{]} If you have an issue when you try to create a {\ttfamily .gitignore} file on Windows, refer to \doxylink{namespace_microsoft}{Microsoft}\textquotesingle{}s documentation on \href{https://docs.microsoft.com/en-us/azure/devops/repos/git/ignore-files?view=azure-devops&tabs=command-line\#create-a-gitignore}{\texttt{ how to create a .gitignore file from the command line}}. \end{quote}

The following template ignores all core Visual Scripting files, but preserves your project settings and variables. It also includes the standard \doxylink{namespace_unity}{Unity} ignore directives for files that you can exclude from version control. For more information, see the {\ttfamily Unity.\+gitignore} file included in \href{https://github.com/github/gitignore/blob/master/Unity.gitignore}{\texttt{ Git\+Hub\textquotesingle{}s gitignore template repository}}.

Refer to the comments in the template for which lines to comment or remove.
