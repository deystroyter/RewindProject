Null nodes deal with the null value, which is scripting lingo for "{}nothing"{}.\hypertarget{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-nulls_autotoc_md2054}{}\doxysection{\texorpdfstring{The null node}{The null node}}\label{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-nulls_autotoc_md2054}
The null node always returns null as a value. Leaving a \doxylink{namespace_unity}{Unity} object reference field empty ("{}\+None"{}) automatically means null.\hypertarget{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-nulls_autotoc_md2055}{}\doxysection{\texorpdfstring{Null Check}{Null Check}}\label{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-nulls_autotoc_md2055}
The null check is a shortcut for a branch on an equality comparison with null. It can be useful to direct the flow in different directions depending on whether a value is null.

For example, it can be used to handle a situation differently whether a transform has a parent in the hierarchy or not.

\hypertarget{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-nulls_autotoc_md2056}{}\doxysection{\texorpdfstring{Null Coalesce}{Null Coalesce}}\label{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-nulls_autotoc_md2056}
The null coalesce node provides a fallback value in case the original input is null.

For example, the null coalesce node defines a default fallback audio clip in case the one on the audio source is missing.
