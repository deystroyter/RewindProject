 You can use the empty graph creation flow to create a new unassigned graph for use in your project.



\begin{quote}
\mbox{[}!\+NOTE\mbox{]} To use the graph file, you must \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-attach-graph-machine}{attach it to a Script Machine or State Machine}{0}. \end{quote}
For more information on other ways to create a graph file, see \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-create-graph}{Create a new graph file}{0}.

To create a new unassigned graph\+:


\begin{DoxyEnumerate}
\item \mbox{[}!include\doxylink{vs-visual-scripting-window_8md}{visual-\/scripting-\/window}\mbox{]}
\item In the new Visual Scripting window, select one of the following options\+:
\begin{DoxyItemize}
\item To create a new Script Graph, select {\bfseries{Create new Script Graph}}.
\item To create a new State Graph, select {\bfseries{Create new State Graph}}.
\end{DoxyItemize}
\item Choose a location to save the new graph file.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Enter a name for the graph.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select {\bfseries{Save}}.

The new graph file automatically opens in a new window.
\end{DoxyEnumerate}

The new graph file should look similar to the following image\+:

After you create a new graph, attach it to a Script Machine or State Machine to use it in your application. For more information, see \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-attach-graph-machine}{Attach a graph file to a Script Machine or State Machine}{0}.
