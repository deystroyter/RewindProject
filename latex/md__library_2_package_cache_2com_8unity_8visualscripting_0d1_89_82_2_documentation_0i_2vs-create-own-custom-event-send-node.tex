After you create a Custom Scripting Event node, you can create a Custom Scripting Event Sender node to trigger the Event from any other Script Graph in the same scene, or the same Script Graph.

You can also choose to create a separate script to trigger the Event from code. After you regenerate your Node Library, the Custom Scripting Event Sender node appears in the fuzzy finder. If you didn\textquotesingle{}t change the {\ttfamily \mbox{[}Unit\+Category\mbox{]}} or {\ttfamily \mbox{[}Unit\+Title\mbox{]}} from the sample code, then the fuzzy finder displays the node under {\bfseries{Events}} \texorpdfstring{$>$}{>} {\bfseries{My\+Events}}, as the {\bfseries{Send My Custom Event}} node.
