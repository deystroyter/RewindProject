\mbox{[}!\+NOTE\mbox{]} The On Mouse Over \mbox{[}!include\doxylink{nodes-note-manual_8md}{nodes-\/note-\/manual}\mbox{]} \end{quote}
The On Mouse Over node listens for a user\textquotesingle{}s mouse to land over a specified Game\+Object\textquotesingle{}s Collider. While the user\textquotesingle{}s mouse is over the Collider, it triggers the next node connected to it once every frame. It doesn\textquotesingle{}t send or receive any other data.

In the following example, the On Mouse Over node triggers a Timer node when the user moves their mouse over the {\bfseries{Target}} Game\+Object. The Timer runs for 2 seconds and triggers a Color Lerp node. For every {\bfseries{Tick}} of the Timer node, the Color Lerp node uses the {\bfseries{Elapsed}} value to calculate a new Color between Color {\bfseries{A}} and Color {\bfseries{B}} to make a smooth transition between colors. The Material Set Color node uses the {\bfseries{Result}} from the Color Lerp node to set a new Color on the {\bfseries{Object}} material.



While the user\textquotesingle{}s mouse is over the {\bfseries{Target}} Game\+Object, the objects that use the {\bfseries{Object}} material in the scene transition from red to blue over two seconds. The transition repeats until the user\textquotesingle{}s mouse leaves the {\bfseries{Target}}\textquotesingle{}s Collider.
