\mbox{[}!\+NOTE\mbox{]} The On Mouse Exit \mbox{[}!include\doxylink{nodes-note-manual_8md}{nodes-\/note-\/manual}\mbox{]} \end{quote}
The On Mouse Exit node listens for the user\textquotesingle{}s mouse pointer location to exit the Collider of a specified Game\+Object. When the mouse exits the Collider or GUI element, the node triggers the next node connected to it. It doesn\textquotesingle{}t send or receive any other data.

In the following example, continued from the example from the \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-nodes-events-on-mouse-enter}{On Mouse Enter node}{0}, the On Mouse Exit node triggers a Destroy Game\+Object node when the user\textquotesingle{}s mouse exits the Collider on the Script Machine\textquotesingle{}s Game\+Object. The Destroy Game\+Object node destroys the Game\+Object assigned to the Spotlight Scene variable. The Game\+Object was created and assigned to the variable elsewhere in the graph.



When the user\textquotesingle{}s mouse leaves the Collider, the {\bfseries{Target}} Game\+Object no longer has a spotlight.
