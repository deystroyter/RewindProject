To use the Input System package with Visual Scripting to capture input in your project\+:


\begin{DoxyEnumerate}
\item \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-open-graph-edit}{Open}{0} or \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-create-graph}{create}{0} a Script Graph attached to the Game\+Object that you want your users to move.
\item \mbox{[}!include\doxylink{vs-open-fuzzy-finder_8md}{open-\/fuzzy-\/finder}\mbox{]}
\item Go to {\bfseries{Events}} \texorpdfstring{$>$}{>} {\bfseries{Input}} or search for {\bfseries{On Input System Event}}.
\item Select an Input System Event node. In this example, select the {\bfseries{On Input System Event Vector 2}} node to add it to the graph.
\item Set the {\bfseries{On Input System Event Vector 2}} node\textquotesingle{}s {\bfseries{Input Action Change Type}} control to {\bfseries{On Hold}}\+:


\item In the {\bfseries{Input Action}} list on the {\bfseries{On Input System Event Vector 2}} node, select an Input Action to trigger the node. In this example, select {\bfseries{Move}}.
\end{DoxyEnumerate}

\begin{quote}
\mbox{[}!\+NOTE\mbox{]} By default, Visual Scripting displays all Input Actions from the Input Action asset attached to your current Game\+Object\textquotesingle{}s Player Input component. \end{quote}

\item \mbox{[}!include\doxylink{vs-open-fuzzy-finder_8md}{open-\/fuzzy-\/finder}\mbox{]}
\end{DoxyEnumerate}

\mbox{[}!\+TIP\mbox{]} If a context menu appears when you right-\/click, select {\bfseries{Add Node}} to open the fuzzy finder. \end{quote}

\begin{DoxyEnumerate}

\item Select {\bfseries{Get X}} to add the {\bfseries{Vector 3 Get X}} node to the graph. ~\newline

\item \mbox{[}!include\doxylink{vs-open-fuzzy-finder_8md}{open-\/fuzzy-\/finder}\mbox{]}
\item Go to {\bfseries{Codebase}} \texorpdfstring{$>$}{>} {\bfseries{\doxylink{namespace_unity}{Unity} Engine}} \texorpdfstring{$>$}{>} {\bfseries{Vector 3}} or search for {\bfseries{Vector 3 Get Z}}.
\item Select {\bfseries{Get Z}} to add the {\bfseries{Vector 3 Get Z}} node to the graph.
\item Select the {\bfseries{Vector 2 Value}} output port on the {\bfseries{On Input System Event Vector 2}} node.
\item \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-creating-connections}{Make a connection}{0} to the {\bfseries{Target}} input port on the {\bfseries{Vector 3 Get X}} node\+:


\item Select the {\bfseries{Vector 2 Value}} output port.
\item \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-creating-connections}{Make a connection}{0} to the {\bfseries{Target}} port on the {\bfseries{Vector 3 Get Z}} node.
\item \mbox{[}!include\doxylink{vs-open-fuzzy-finder_8md}{open-\/fuzzy-\/finder}\mbox{]}
\item Go to {\bfseries{Codebase}} \texorpdfstring{$>$}{>} {\bfseries{\doxylink{namespace_unity}{Unity} Engine}} \texorpdfstring{$>$}{>} {\bfseries{Transform}} or search for {\bfseries{Translate}}.
\item Select {\bfseries{Transform\+: Translate (X, Y, Z)}} to add the Translate node to the graph.
\item Select the {\bfseries{Value\+: Float}} output port on the {\bfseries{Vector 3 Get X}} node.
\item \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-creating-connections}{Make a connection}{0} to the {\bfseries{X}} float input port on the {\bfseries{Translate}} node.
\item Select the {\bfseries{Value\+: Float}} output port on the {\bfseries{Vector 3 Get X}} node.
\item \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-creating-connections}{Make a connection}{0} to the {\bfseries{Z}} float input port on the {\bfseries{Translate}} node.
