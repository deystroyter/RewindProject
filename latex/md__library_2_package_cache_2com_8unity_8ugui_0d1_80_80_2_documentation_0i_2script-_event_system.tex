 This subsystem is responsible for controlling all the other elements that make up eventing. It coordinates which Input Module is currently active, which Game\+Object is currently considered \textquotesingle{}selected\textquotesingle{}, and a host of other high level Event System concepts.

Each \textquotesingle{}Update\textquotesingle{} the Event System receives the call, looks through its Input Modules and figures out which is the Input Module that should be used for this tick.
