 Use the Inspector window to change the playable asset properties of a Control clip. To view the playable asset properties for a Control clip, select a Control clip in the \doxylink{namespace_timeline}{Timeline} window and expand {\bfseries{Control Playable Asset}} in the Inspector window.



{\itshape Inspector window showing the {\bfseries{Control Playable Asset}} properties for the selected Control clip}\hypertarget{md__library_2_package_cache_2com_8unity_8timeline_0d1_87_86_2_documentation_0i_2insp__clp__ctrl__plyb_autotoc_md1176}{}\doxysection{\texorpdfstring{Source Game Object}{Source Game Object}}\label{md__library_2_package_cache_2com_8unity_8timeline_0d1_87_86_2_documentation_0i_2insp__clp__ctrl__plyb_autotoc_md1176}
Use {\bfseries{Source Game Object}} to select the Game\+Object with the Particle System, nested \doxylink{namespace_timeline}{Timeline} instance, or ITime\+Control Script for the selected Control clip. Changing the {\bfseries{Source Game Object}} changes what the Control clip controls.\hypertarget{md__library_2_package_cache_2com_8unity_8timeline_0d1_87_86_2_documentation_0i_2insp__clp__ctrl__plyb_autotoc_md1177}{}\doxysection{\texorpdfstring{Prefab}{Prefab}}\label{md__library_2_package_cache_2com_8unity_8timeline_0d1_87_86_2_documentation_0i_2insp__clp__ctrl__plyb_autotoc_md1177}
Use {\bfseries{Prefab}} to select a Prefab to instantiate when the \doxylink{namespace_timeline}{Timeline} instance plays in Play Mode. When a Prefab is selected, the label of the {\bfseries{Source Game Object}} property changes to {\bfseries{Parent Object}}.

When in Play Mode, the Prefab is instantiated as a child of the {\bfseries{Parent Object}}. Although the Prefab is instantiated at the start of the \doxylink{namespace_timeline}{Timeline} instance, the Prefab is only activated during the Control clip. When the Control clip ends, the Prefab instance is deactivated.\hypertarget{md__library_2_package_cache_2com_8unity_8timeline_0d1_87_86_2_documentation_0i_2insp__clp__ctrl__plyb_autotoc_md1178}{}\doxysection{\texorpdfstring{Control Activation}{Control Activation}}\label{md__library_2_package_cache_2com_8unity_8timeline_0d1_87_86_2_documentation_0i_2insp__clp__ctrl__plyb_autotoc_md1178}
Enable {\bfseries{Control Activation}} to activate the {\bfseries{Source Game Object}} while the Control clip plays. Disable this property to activate the {\bfseries{Source Game Object}} during the entire \doxylink{namespace_timeline}{Timeline} instance.

The {\bfseries{Control Activation}} property only affects Control clips that control a nested \doxylink{namespace_timeline}{Timeline} instance or a Particle System.\hypertarget{md__library_2_package_cache_2com_8unity_8timeline_0d1_87_86_2_documentation_0i_2insp__clp__ctrl__plyb_autotoc_md1179}{}\doxysection{\texorpdfstring{Post Playback}{Post Playback}}\label{md__library_2_package_cache_2com_8unity_8timeline_0d1_87_86_2_documentation_0i_2insp__clp__ctrl__plyb_autotoc_md1179}
When {\bfseries{Control Activation}} is enabled, use the {\bfseries{Post Playback}} property to set the activation state for the nested \doxylink{namespace_timeline}{Timeline} instance when the main \doxylink{namespace_timeline}{Timeline} stops playing. The {\bfseries{Post Playback}} property only affects nested \doxylink{namespace_timeline}{Timeline} instances.
