 The Version Control package provides an integration of \doxylink{namespace_unity}{Unity} Version Control (\doxylink{namespace_unity}{Unity} VCS, formerly Plastic SCM) in the \doxylink{namespace_unity}{Unity} Editor.

\doxylink{namespace_unity}{Unity} Version Control enables you to work collaboratively by providing advanced features such as branching, locking, merging, and a standalone Desktop GUI.

Learn more about \href{https://unity.com/solutions/version-control}{\texttt{ Unity Version Control}}.

\item To start from an existing \doxylink{namespace_unity}{Unity} Version Control repository, see \doxysectlink{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_get_started_existing_repository}{Get started with an existing repository}{0}.
\end{DoxyItemize}

For more information on how to get started, refer to the \href{https://docs.unity.com/ugs/en-us/manual/devops/manual/unity-version-control}{\texttt{ Unity Version Control documentation}}.
