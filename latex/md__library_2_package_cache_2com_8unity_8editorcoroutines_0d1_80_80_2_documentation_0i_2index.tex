 The Editor Coroutines package allows the user to start the execution of \href{https://docs.microsoft.com/en-us/dotnet/csharp/language-reference/keywords/yield}{\texttt{ iterator methods}} within the Editor similar to how we handle \href{https://docs.unity3d.com/Manual/Coroutines.html}{\texttt{ Coroutines}} inside \href{https://docs.unity3d.com/ScriptReference/MonoBehaviour.html}{\texttt{ Mono\+Behaviour}} scripts during runtime.\hypertarget{md__library_2_package_cache_2com_8unity_8editorcoroutines_0d1_80_80_2_documentation_0i_2index_autotoc_md192}{}\doxysection{\texorpdfstring{Installing Editor Coroutines}{Installing Editor Coroutines}}\label{md__library_2_package_cache_2com_8unity_8editorcoroutines_0d1_80_80_2_documentation_0i_2index_autotoc_md192}
To install this package, follow the instructions in the \href{https://docs.unity3d.com/Packages/com.unity.package-manager-ui@latest/index.html}{\texttt{ Package Manager documentation}}.

\begin{quote}
{\bfseries{Note}}\+: While this package is in preview, the Package Manager needs to be configured to show {\bfseries{Preview \doxylink{namespace_packages}{Packages}}}. (Under the {\bfseries{Advanced}} drop-\/down menu, enable {\bfseries{Show preview packages}}.) Then search for the Editor Coroutines package. \end{quote}

\begin{DoxyItemize}
\item 2018.\+1 and later (recommended)
\end{DoxyItemize}

\begin{quote}
{\bfseries{Note}}\+: If you install the Memory Profiler package it will automatically install the Editor Coroutines package as a dependency. \end{quote}
\hypertarget{md__library_2_package_cache_2com_8unity_8editorcoroutines_0d1_80_80_2_documentation_0i_2index_autotoc_md196}{}\doxysubsection{\texorpdfstring{Known limitations}{Known limitations}}\label{md__library_2_package_cache_2com_8unity_8editorcoroutines_0d1_80_80_2_documentation_0i_2index_autotoc_md196}
Editor Coroutines version 0.\+0.\+1-\/preview.\+2 includes the following known limitation(s)\+:

The iterator functions passed to Editor Coroutines do not support yielding any of the instruction classes present inside the \doxylink{namespace_unity}{Unity} Scripting API (e.\+g., \href{https://docs.unity3d.com/ScriptReference/WaitForSeconds.html}{\texttt{ Wait\+For\+Seconds}}, \href{https://docs.unity3d.com/ScriptReference/WaitForEndOfFrame.html}{\texttt{ Wait\+For\+End\+Of\+Frame}}), except for the \href{https://docs.unity3d.com/ScriptReference/CustomYieldInstruction.html}{\texttt{ Custom\+Yield\+Instruction}} derived classes with the {\ttfamily Move\+Next} method implemented.

\begin{quote}
{\bfseries{Tip}}\+: {\ttfamily yield return null} is a way to skip a frame within the Editor. \end{quote}
