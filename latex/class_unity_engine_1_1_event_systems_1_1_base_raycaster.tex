\doxysubsection{Detailed Description}
Base class for any Ray\+Caster. 

A Raycaster is responsible for raycasting against scene elements to determine if the cursor is over them. Default Raycasters include Physics\+Raycaster, Physics2\+DRaycaster, Graphic\+Raycaster. Custom raycasters can be added by extending this class.
