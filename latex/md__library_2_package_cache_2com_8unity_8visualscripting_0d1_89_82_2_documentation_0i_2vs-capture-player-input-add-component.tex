 To use the Input System package with Visual Scripting, add a Player Input component to the same Game\+Object as the Script Graph and create an Input Actions asset.

You must add the Player Input component and create the Input Actions asset before you create the Script Graph.

\begin{quote}
\mbox{[}!\+NOTE\mbox{]} If the Input System package isn\textquotesingle{}t installed in your project, follow the Input System documentation\textquotesingle{}s \href{https://docs.unity3d.com/Packages/com.unity.inputsystem@latest/index.html?subfolder=/manual/Installation.html}{\texttt{ Installation guide}} to install the package. Go to {\bfseries{Window}} \texorpdfstring{$>$}{>} {\bfseries{Package Manager}} to check your installed packages. \end{quote}
To add a Player Input component to a Game\+Object\+:


\begin{DoxyEnumerate}
\item \mbox{[}!include\doxylink{vs-open-hierarchy-window_8md}{open-\/hierarchy-\/window}\mbox{]}
\item In the Hierarchy window, select the Game\+Object that you want to move with the Script Graph.
\item \mbox{[}!include\doxylink{vs-open-inspector-window_8md}{open-\/inspector-\/window}\mbox{]}
\item Select {\bfseries{Add Component}}. The Components menu opens.
\item In the Components menu, do one of the following\+:
\begin{DoxyItemize}
\item Go to {\bfseries{Input}}.
\item In the Search bar, enter {\ttfamily Player Input}.
\end{DoxyItemize}
\item Select the {\bfseries{Player Input}} component to add it to the Game\+Object.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Add an Input Actions asset to the Player Input component. Do one of the following\+:
\begin{DoxyItemize}
\item Create a new Input Actions asset.
\item Use an existing Input Actions asset.
\end{DoxyItemize}
\end{DoxyEnumerate}

\item Do one of the following\+:
\begin{DoxyItemize}
\item Click the {\bfseries{Actions}} field\textquotesingle{}s object picker (circle icon) and in the {\bfseries{Select\+Input\+Action\+Asset}} window, select the asset.
\item Click and drag a file from your Project window and drop it into the {\bfseries{Actions}} field.
\end{DoxyItemize}
