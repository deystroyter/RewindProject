 Type Options specify which node inputs and outputs Visual Scripting supports. After you \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-add-remove-node-library}{add a new assembly}{0}, you must add types specific to those nodes to your Type Options. Add types to make the nodes and their types accessible in the \href{vs-interface-overview.md\#the-fuzzy-finder}{\texttt{ fuzzy finder}} and \href{vs-interface-overview.md\#the-blackboard}{\texttt{ the Blackboard}}.

You can\textquotesingle{}t use a node that has an input or output type that isn\textquotesingle{}t listed in your Type Options.\hypertarget{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-add-remove-type-options_autotoc_md1661}{}\doxysection{\texorpdfstring{Add a type to your Type Options}{Add a type to your Type Options}}\label{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-add-remove-type-options_autotoc_md1661}
To add a new type to your Type Options list\+:


\begin{DoxyEnumerate}
\item \mbox{[}!include\doxylink{vs-open-project-settings_8md}{open-\/project-\/settings}\mbox{]}
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select {\bfseries{Visual Scripting}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Expand {\bfseries{Type Options}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item At the end of the types list, select {\bfseries{Add}} (+).
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item In the new type entry, select \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}(No Type)\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} to open the {\bfseries{Type}} menu.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select an available type from the {\bfseries{Type}} menu.
\end{DoxyEnumerate}

Visual Scripting adds the new type to your Type Options. To remove a type from your Type Options list\+:


\begin{DoxyEnumerate}
\item \mbox{[}!include\doxylink{vs-open-project-settings_8md}{open-\/project-\/settings}\mbox{]}
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select {\bfseries{Visual Scripting}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Expand {\bfseries{Type Options}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item In the types list, locate the entry for the type you want to remove.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select {\bfseries{Remove}} (-\/).
\end{DoxyEnumerate}

Visual Scripting removes the type from your Type Options. To make sure that your change appears in your project, \href{vs-configuration.md\#Regen}{\texttt{ regenerate your Node Library}}.
