In \doxylink{namespace_unity}{Unity}, you use a project to design and develop a game. A project stores all of the files related to a game, such as the asset and {\bfseries{Scene}} files. See \href{https://docs.unity3d.com/Manual/2Dor3D.html}{\texttt{ 2D or 3D projects}}.\hypertarget{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_glossary_autotoc_md139}{}\doxysubsection{\texorpdfstring{Version Control}{Version Control}}\label{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_glossary_autotoc_md139}
A system for managing changes to a set of files. You can use \doxylink{namespace_unity}{Unity} in conjunction with most {\bfseries{version control}} tools, including {\bfseries{\doxylink{namespace_unity}{Unity} Version Control}} and {\bfseries{Perforce}}. See \href{https://docs.unity3d.com/Manual/VersionControl.html}{\texttt{ Version Control}}.\hypertarget{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_glossary_autotoc_md140}{}\doxysubsection{\texorpdfstring{Ignore file}{Ignore file}}\label{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_glossary_autotoc_md140}
A special file used in many {\bfseries{Version Control}} Systems which specifies files to be excluded from {\bfseries{version control}}. In \doxylink{namespace_unity}{Unity} projects, several folders should be excluded from {\bfseries{version control}}.\hypertarget{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_glossary_autotoc_md141}{}\doxysubsection{\texorpdfstring{Repository}{Repository}}\label{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_glossary_autotoc_md141}
A shared history of changes made to the project\textquotesingle{}s files, saved on a server.\hypertarget{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_glossary_autotoc_md142}{}\doxysubsection{\texorpdfstring{Workspace}{Workspace}}\label{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_glossary_autotoc_md142}
Your local copy of the repository, interacting with the version control system. It\textquotesingle{}s where you download the project\textquotesingle{}s files, make the required changes and perform checkins.\hypertarget{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_glossary_autotoc_md143}{}\doxysubsection{\texorpdfstring{Check-\/in}{Check-\/in}}\label{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_glossary_autotoc_md143}

You can enter a comment before checking in your changes.\hypertarget{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_glossary_autotoc_md144}{}\doxysection{\texorpdfstring{Unity Version Control terms}{Unity Version Control terms}}\label{md__library_2_package_cache_2com_8unity_8collab-proxy_0d2_83_81_2_documentation_0i_2_glossary_autotoc_md144}
