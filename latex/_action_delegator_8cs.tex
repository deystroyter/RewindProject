class {\bfseries Unity\+Engine.\+Test\+Tools.\+NUnit\+Extensions.\+Action\+Delegator}
\begin{DoxyCompactList}\small\item\em This class delegates actions from the NUnit thread that should be executed on the main thread. NUnit thread calls Delegate which blocks the execution on the thread until the action is executed. The main thread will poll for awaiting actions (Has\+Action) and invoke them (Execute). Once the action is executed, the main thread releases the lock and executino on the NUnit thread is continued. \end{DoxyCompactList}\end{DoxyCompactItemize}
