 The Version Control package provides an integration of \doxylink{namespace_unity}{Unity} Version Control (\doxylink{namespace_unity}{Unity} VCS, formerly Plastic SCM) in the \doxylink{namespace_unity}{Unity} Editor.

It is installed by default with the Editor, and follows the \doxylink{namespace_unity}{Unity} support schedule. Currently, the minimum supported version of the \doxylink{namespace_unity}{Unity} Editor is 2021.\+3 LTS.
