\doxysubsection{Detailed Description}
Interface to navigate through Timelines and Sub\+Timelines for the \doxylink{namespace_timeline}{Timeline} window. 

Timeline\+Navigator gives you access to the \doxylink{namespace_timeline}{Timeline} window breadcrumbs functionality. Use it to programmatically dig into Sub\+Timelines, navigate to parent Timelines or navigate \doxylink{namespace_timeline}{Timeline} Window breadcrumbs.

The Sequence\+Context provided must be a valid navigation destination.

Valid navigation destinations\+:
\begin{DoxyItemize}
\item The parent context returned by Get\+Parent\+Context.
\item The root context returned by Get\+Root\+Context.
\item Any Sequence\+Context returned by Get\+Child\+Contexts.
\item Any Sequence\+Context returned by Get\+Breadcrumbs.
\end{DoxyItemize}

Note\+: This method cannot be used to change the root Sequence\+Context. To change the root Sequence\+Context, use Timeline\+Editor\+Window.\+Set\+Timeline.
