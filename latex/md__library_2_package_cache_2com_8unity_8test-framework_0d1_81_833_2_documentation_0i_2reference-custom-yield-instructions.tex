\item When creating an Editor test that uses the {\ttfamily Unity\+Test} attribute, use this to trigger the Editor to enter Play Mode.
\item Throws an exception if the Editor is already in Play Mode or if there is a \href{https://support.unity3d.com/hc/en-us/articles/205930539-How-do-I-interpret-a-compiler-error-}{\texttt{ script compilation error}}.
