\doxysubsection{Detailed Description}
The Test\+Runner\+Api retrieves and runs tests programmatically from code inside the project, or inside other packages. Test\+Runner\+Api is a \href{https://docs.unity3d.com/ScriptReference/ScriptableObject.html}{\texttt{ Scriptable\+Object}}. You can initialize the API like this\+: 

{\ttfamily  var test\+Runner\+Api = Scriptable\+Object.\+Create\+Instance\texorpdfstring{$<$}{<}Test\+Runner\+Api\texorpdfstring{$>$}{>}(); } Note\+: You can subscribe and receive test results in one instance of the API, even if the run starts from another instance.
