 You can use the empty graph creation flow to create a new graph file and assign it to an existing Game\+Object in your project.



For more information on other ways to create a graph file, see \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-create-graph}{Create a new graph file}{0}.\hypertarget{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-create9cd56cfb3a8d150ca0f48b41ff1af66f_autotoc_md1772}{}\doxysection{\texorpdfstring{Create a Script Graph}{Create a Script Graph}}\label{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-create9cd56cfb3a8d150ca0f48b41ff1af66f_autotoc_md1772}
To create a new Script Graph and assign it to an existing Game\+Object\+:


\begin{DoxyEnumerate}
\item \mbox{[}!include\doxylink{vs-open-hierarchy-window_8md}{open-\/hierarchy-\/window}\mbox{]}
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item In the Hierarchy window, select the Game\+Object where you want to assign the new graph.
\item \mbox{[}!include\doxylink{vs-visual-scripting-window_8md}{visual-\/scripting-\/window}\mbox{]}
\item Expand {\bfseries{Create new Script Graph}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select {\bfseries{on selected game object}}.
\item Choose a location to save the new graph file.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Enter a name for the graph.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select {\bfseries{Save}}.

The new graph file automatically opens in a new window.
\end{DoxyEnumerate}
