 {\ttfamily Mono\+Behaviour\+Test} is a \href{https://docs.unity3d.com/ScriptReference/Coroutine.html}{\texttt{ coroutine}} and a helper for writing \href{https://docs.unity3d.com/ScriptReference/MonoBehaviour.html}{\texttt{ Mono\+Behaviour}} tests.

Yield a {\ttfamily Mono\+Behaviour\+Test} when using the {\ttfamily Unity\+Test} attribute to instantiate the {\ttfamily Mono\+Behaviour} you wish to test and wait for it to finish running. Implement the {\ttfamily IMono\+Behaviour\+Test} interface on the {\ttfamily Mono\+Behaviour} to state when the test completes.\hypertarget{md__library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2reference-tests-monobehaviour_autotoc_md706}{}\doxysection{\texorpdfstring{Example}{Example}}\label{md__library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2reference-tests-monobehaviour_autotoc_md706}
