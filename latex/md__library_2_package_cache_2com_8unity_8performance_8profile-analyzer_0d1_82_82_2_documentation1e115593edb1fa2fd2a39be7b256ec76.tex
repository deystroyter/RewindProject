 This workflow explains how to compare two frames from the same data set. In this example, it explains how to compare the median and longest frames in a data set. Comparing the median and longest frames is useful to help understand what is happening in the longest frame that is not happening in an average frame, or what is taking longer than average to complete.\hypertarget{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation1e115593edb1fa2fd2a39be7b256ec76_autotoc_md396}{}\doxysection{\texorpdfstring{Step 1\+: Collect performance data to analyze}{Step 1\+: Collect performance data to analyze}}\label{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation1e115593edb1fa2fd2a39be7b256ec76_autotoc_md396}
Open the Profile Analyzer window (menu\+: {\bfseries{Window \texorpdfstring{$>$}{>} Analysis \texorpdfstring{$>$}{>} Profile Analyzer}}) and collect some profiling data. To pull data from an active profiling session, click the {\bfseries{Pull Data}} button. This pulls in the current set of available frames from the Profiler. If you don\textquotesingle{}t have an active profile session, click the {\bfseries{Open Profiler Window}} button, then load or record some data.

For more information on how to collect data, see the workflow documentation on \doxysectlink{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4}{Collecting and viewing data}{0}.\hypertarget{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation1e115593edb1fa2fd2a39be7b256ec76_autotoc_md397}{}\doxysection{\texorpdfstring{Step 2\+: Open Compare view}{Step 2\+: Open Compare view}}\label{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation1e115593edb1fa2fd2a39be7b256ec76_autotoc_md397}
Click the {\bfseries{Compare}} button in the toolbar to switch to the \doxysectlink{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation_0i_2compare-view}{Compare View}{0}.\hypertarget{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation1e115593edb1fa2fd2a39be7b256ec76_autotoc_md398}{}\doxysection{\texorpdfstring{Step 3\+: Select the median and longest frames}{Step 3\+: Select the median and longest frames}}\label{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation1e115593edb1fa2fd2a39be7b256ec76_autotoc_md398}
