\doxysubsection{Detailed Description}
A 2D rectangular mask that allows for clipping / masking of areas outside the mask. 

The Rect\+Mask2D behaves in a similar way to a standard Mask component. It differs though in some of the restrictions that it has. A Rect\+Mask2D\+: \texorpdfstring{$\ast$}{*}\+Only works in the 2D plane \texorpdfstring{$\ast$}{*}\+Requires elements on the mask to be coplanar. \texorpdfstring{$\ast$}{*}\+Does not require stencil buffer / extra draw calls \texorpdfstring{$\ast$}{*}\+Requires fewer draw calls \texorpdfstring{$\ast$}{*}\+Culls elements that are outside the mask area.
