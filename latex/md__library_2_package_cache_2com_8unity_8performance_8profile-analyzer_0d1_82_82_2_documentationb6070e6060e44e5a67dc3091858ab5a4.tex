 This workflow explains how to populate the \href{https://docs.unity3d.com/Manual/Profiler.html}{\texttt{ Unity Profiler}} and Profile Analyzer with data. Note that the Profile Analyzer only analyzes CPU data from the Profiler.\hypertarget{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4_autotoc_md383}{}\doxysection{\texorpdfstring{Step 1\+: Open the Unity Profiler}{Step 1\+: Open the Unity Profiler}}\label{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4_autotoc_md383}
Go to the menu\+: {\bfseries{Window \texorpdfstring{$>$}{>} Analysis \texorpdfstring{$>$}{>} Profiler}} or press Ctrl+7 (Command+7 on mac\+OS). If you have the Profile Analyzer window open, you can click the {\bfseries{Open Profiler Window}} button.\hypertarget{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4_autotoc_md384}{}\doxysection{\texorpdfstring{Step 2\+: Populate the Profiler with data}{Step 2\+: Populate the Profiler with data}}\label{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4_autotoc_md384}
To use the Profile Analyzer, you must populate it with data from the Profiler. To add data to the Profiler, you can either record some new data, or load a Profiler capture file in the .data file format.\hypertarget{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4_autotoc_md385}{}\doxysubsection{\texorpdfstring{a) Record new data}{a) Record new data}}\label{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4_autotoc_md385}
Click the {\bfseries{Attach to Player}} dropdown at the top of the window (next to the Record button) and select a player to profile. By default this is set to {\bfseries{Playmode}}. Click the {\bfseries{Record}} button to start recording data. If you enabled {\bfseries{Autoconnect to Profiler}} in the Build Settings, the Profiler automatically collects data when you start a built player.

For more information on how to record data in the Profiler, see the documentation on \href{https://docs.unity3d.com/Manual/profiler-profiling-applications.html}{\texttt{ Profiling your application}} in the \doxylink{namespace_unity}{Unity} User Manual.\hypertarget{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4_autotoc_md386}{}\doxysubsection{\texorpdfstring{b) Load data}{b) Load data}}\label{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4_autotoc_md386}
To load a saved .data file, in the top right of the Profiler window, select the {\bfseries{Load}} button.\hypertarget{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4_autotoc_md387}{}\doxysection{\texorpdfstring{Step 3\+: Pull the data into the Profile Analyzer window}{Step 3\+: Pull the data into the Profile Analyzer window}}\label{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4_autotoc_md387}
Open the Profile Analyzer window (menu\+: {\bfseries{Window \texorpdfstring{$>$}{>} Analysis \texorpdfstring{$>$}{>} Profile Analyzer}}) and then select the {\bfseries{Pull Data}} button in the \doxysectlink{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentation_0i_2frame-range-selection}{Frame Control}{0} pane. The Profile Analyzer then pulls in the data that is loaded in the Profiler window.

\texorpdfstring{$>$}{>}\mbox{[}!\+TIP\mbox{]} \texorpdfstring{$>$}{>}The Profiler window and the Profile Analyzer window require a lot of screen real-\/estate. Docking the two windows together in a single tabbed window lets you navigate between the two views quickly.

{\itshape The Profile Analyzer in the Compare mode docked next to the Profiler in one window}\hypertarget{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4_autotoc_md388}{}\doxysection{\texorpdfstring{Step 4\+: Load and save Profile Analyzer data}{Step 4\+: Load and save Profile Analyzer data}}\label{md__library_2_package_cache_2com_8unity_8performance_8profile-analyzer_0d1_82_82_2_documentationb6070e6060e44e5a67dc3091858ab5a4_autotoc_md388}
To save the data from the Profile Analyzer, click the {\bfseries{Save}} button in any view. Select where you would like to save your data, and then \doxylink{namespace_unity}{Unity} saves the data in the .pdata format. To load this data, click the {\bfseries{Load}} button in any view.

\texorpdfstring{$>$}{>}\mbox{[}!\+NOTE\mbox{]} \texorpdfstring{$>$}{>}When you load data into the Profile Analyzer, the data must be in the Profile Analyzer .pdata format. If you have data from the Profiler in the .data file format, open it in the Profiler first, and in the Profile Analyzer select the {\bfseries{Pull Data}} button.
