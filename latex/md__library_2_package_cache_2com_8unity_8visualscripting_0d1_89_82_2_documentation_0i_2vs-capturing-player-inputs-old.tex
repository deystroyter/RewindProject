To create a basic Script Graph that uses the Input Manager to capture input\+:


\begin{DoxyEnumerate}
\item 

\href{vs-open-graph-edit.md}{\texttt{ Open}} or \href{vs-create-graph.md}{\texttt{ create}} a Script Graph attached to the Game\+Object that you want your users to move.


\item 

If there isn\textquotesingle{}t an {\bfseries{On Update}} or similar Event node in your graph\+:


\begin{DoxyEnumerate}[label=\enumalphalphcnt*]

\item 

\mbox{[}!include\doxylink{vs-open-fuzzy-finder_8md}{open-\/fuzzy-\/finder}\mbox{]}


\item 

Go to {\bfseries{Events}} \texorpdfstring{$>$}{>} {\bfseries{Lifecycle}}, or enter {\bfseries{On Update}} in the search field.


\item 

Select the {\bfseries{On Update}} Event node to add it to the graph.


\end{DoxyEnumerate}
\item 

\mbox{[}!include\doxylink{vs-open-fuzzy-finder_8md}{open-\/fuzzy-\/finder}\mbox{]}

\doxysubparagraph*{TIP}



If you right-\/click and the context menu appears, select {\bfseries{Add Node}} to open the fuzzy finder.

 
\item 

Go to {\bfseries{Codebase}} \texorpdfstring{$>$}{>} {\bfseries{\doxylink{namespace_unity}{Unity} Engine}} \texorpdfstring{$>$}{>} {\bfseries{Input}}, or enter {\bfseries{Get Axis}} in the search field.


\item 

Select {\bfseries{Get Axis (Axis Name)}} to add the Get Axis node to the graph.


\item 

Repeat Steps 3 through 5 to create a second {\bfseries{Get Axis (Axis Name)}} node.


\item 

On the first Get Axis node, in the {\bfseries{Axis Name}} input field, enter {\ttfamily Horizontal}.


\item 

On the second Get Axis node, in the {\bfseries{Axis Name}} input field, enter {\ttfamily Vertical}.

\doxysubparagraph*{NOTE}



If an Axis Name doesn\textquotesingle{}t match the name in the Input Manager\textquotesingle{}s Project Settings, Visual Scripting displays an error in the Graph Inspector. When you enter Play mode, the \doxylink{namespace_unity}{Unity} Editor also displays an error in the Console window.

 
\item 

\mbox{[}!include\doxylink{vs-open-fuzzy-finder_8md}{open-\/fuzzy-\/finder}\mbox{]}


\item 

Go to {\bfseries{Codebase}} \texorpdfstring{$>$}{>} {\bfseries{\doxylink{namespace_unity}{Unity} Engine}} \texorpdfstring{$>$}{>} {\bfseries{Transform}} or search for {\bfseries{Translate}}.


\item 

Select {\bfseries{Translate (X, Y, Z)}} to add a Translate node to the graph.


\item 

Select the {\bfseries{Result}} float output port on the {\ttfamily Horizontal} Get Axis node.


\item 

\href{vs-creating-connections.md}{\texttt{ Make a connection}} to the {\bfseries{X}} input port on the {\bfseries{Translate}} node.


\item 

Select the {\bfseries{Result}} float output port on the {\ttfamily Vertical} Get Axis node.


\item 

\href{vs-creating-connections.md}{\texttt{ Make a connection}} to the {\bfseries{Z}} input port on the {\bfseries{Translate}} node. ~\newline
The finished graph looks similar to the following image\+:






\item 

To enter Play mode, select {\bfseries{Play}} from the \href{https://docs.unity3d.com/Manual/Toolbar.html}{\texttt{ Unity Editor\textquotesingle{}s Toolbar}}.


\item 

While in the \href{https://docs.unity3d.com/Manual/GameView.html}{\texttt{ Game view}}, press a key mapped as a {\bfseries{Negative Button}} or {\bfseries{Positive Button}} from the \href{https://docs.unity3d.com/Documentation/Manual/class-InputManager.html}{\texttt{ Input Manager\textquotesingle{}s virtual axes}}.


\end{DoxyEnumerate}
