 You can open a graph file from multiple locations, based on the graph type and its source type.

For information on how to create a new graph file, see \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-create-graph}{Create a new graph file}{0}.\hypertarget{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-open-graph-edit_autotoc_md2058}{}\doxysection{\texorpdfstring{From the Project window}{From the Project window}}\label{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-open-graph-edit_autotoc_md2058}
To open a graph from the Project window\+:


\begin{DoxyEnumerate}
\item \mbox{[}!include\doxylink{vs-open-project-window_8md}{open-\/project-\/window}\mbox{]}
\item Find the location in your Project window\textquotesingle{}s folders where you saved the graph file you want to edit.


\item Double-\/click the graph file to open it in the Graph window.
\end{DoxyEnumerate}\hypertarget{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-open-graph-edit_autotoc_md2059}{}\doxysection{\texorpdfstring{From the Graph Inspector}{From the Graph Inspector}}\label{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-open-graph-edit_autotoc_md2059}
If you have a nested or embedded graph inside another graph file, you can open it from the Graph Inspector.


\begin{DoxyEnumerate}
\item In the Graph window, select the node that represents the graph you want to edit. This node could be a transition, Super State, Subgraph, or State Unit.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Open the \href{vs-interface-overview.md\#the-graph-inspector}{\texttt{ Graph Inspector}}.
\item In the Graph Inspector, select {\bfseries{Edit Graph}}. The graph opens in the same Graph window.
\end{DoxyEnumerate}

\hypertarget{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-open-graph-edit_autotoc_md2060}{}\doxysection{\texorpdfstring{From a Script Machine or State Machine}{From a Script Machine or State Machine}}\label{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-open-graph-edit_autotoc_md2060}
If you\textquotesingle{}ve attached or embedded a graph in a Script Machine or State Machine on a Game\+Object, you can open the graph from the component on the Game\+Object\+:


\begin{DoxyEnumerate}
\item \mbox{[}!include\doxylink{vs-open-hierarchy-window_8md}{open-\/hierarchy-\/window}\mbox{]}
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item In the Hierarchy window, select the Game\+Object that has the Script Machine or State Machine with the graph you want to edit. ~\newline

\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item \mbox{[}!include\doxylink{vs-open-inspector-window_8md}{open-\/inspector-\/window}\mbox{]}
\item On the Script Machine or State Machine component, select {\bfseries{Edit Graph}}.
\end{DoxyEnumerate}
