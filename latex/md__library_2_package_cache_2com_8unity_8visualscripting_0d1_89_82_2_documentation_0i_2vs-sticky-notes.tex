 Use Sticky Notes to add comments to a graph. Sticky Notes can\+:


\begin{DoxyItemize}
\item Describe how a section of your graph works.
\item Leave a note for yourself or others who might work on your \doxylink{namespace_unity}{Unity} project.
\end{DoxyItemize}

You can add text to the title and body of a Sticky Note. You can use Sticky Notes in Script Graphs and State Graphs.
