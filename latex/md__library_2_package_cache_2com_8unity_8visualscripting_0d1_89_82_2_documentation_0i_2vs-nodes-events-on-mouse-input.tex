In the following example, the On Mouse Input node listens for the user to hold the right mouse button and triggers an Instantiate Camera node. The Instantiate node clones the camera saved as the {\bfseries{Camera1}} Scene variable and assigns it to the {\bfseries{New\+Camera}} Scene variable. It sets a new position for the cloned camera with a Transform Set Position node, before it switches which camera renders in the Game view with the Camera Render node.



When the application runs, the default view in the Game view displays all three spheres in the scene.



When the user holds the right mouse button and triggers the On Mouse Input node, the Game view changes to focus on the middle sphere.
