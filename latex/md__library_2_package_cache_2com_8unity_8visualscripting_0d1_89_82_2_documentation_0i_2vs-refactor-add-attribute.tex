\chapter{Add the Renamed\+From attribute to a C\# script}
\hypertarget{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-refactor-add-attribute}{}\label{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-refactor-add-attribute}\index{Add the RenamedFrom attribute to a C\# script@{Add the RenamedFrom attribute to a C\# script}}
\label{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-refactor-add-attribute_autotoc_md2062}%
\Hypertarget{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-refactor-add-attribute_autotoc_md2062}%
 To use nodes generated from a custom C\# script in a project after you rename a member, class, struct, type, or enum, add the {\ttfamily \mbox{[}Renamed\+From\mbox{]}} attribute to the relevant API element in the script file. For more information on the {\ttfamily \mbox{[}Renamed\+From\mbox{]}} attribute, see \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-refactoring}{Refactor a C\# script with Visual Scripting}{0}.

To add the attribute to a C\# script\+:


\begin{DoxyEnumerate}
\item 

\mbox{[}!include\doxylink{vs-open-project-window_8md}{open-\/project-\/window}\mbox{]}


\item 

In the Project window, double-\/click the C\# script file you want to refactor. \doxylink{namespace_unity}{Unity} opens the file in the program you specified in your preferences, under {\bfseries{External Script Editor}}.

\doxysubparagraph*{NOTE}



For more information on script editors in \doxylink{namespace_unity}{Unity}, see \href{https://docs.unity3d.com/Manual/ScriptingToolsIDEs.html}{\texttt{ Integrated development environment (IDE) support}} in the \doxylink{namespace_unity}{Unity} User Manual.

  
\item 

In your external editor, do the following\+:


\begin{DoxyEnumerate}[label=\enumalphalphcnt*]

\item 

Add the {\ttfamily \mbox{[}Renamed\+From\mbox{]}} attribute above the definition of the part of the script you want to rename.


\item 

Add the element\textquotesingle{}s old name as a string to the {\ttfamily \mbox{[}Renamed\+From\mbox{]}} attribute, as its parameter. For example\+:


\begin{DoxyPre}{\ttfamily 
using \doxylink{namespace_unity_engine}{UnityEngine}; 
using \doxylink{namespace_unity_1_1_visual_scripting}{Unity.VisualScripting}; 

[RenamedFrom("{}Character"{})]
public class Player : MonoBehaviour
\{
    [RenamedFrom("{}InflictDamage"{})]
    public void TakeDamage(int damage)
    \{
        //...
    \}
\}
}\end{DoxyPre}
 
\end{DoxyEnumerate}
\item 

\mbox{[}!include\doxylink{vs-save-script_8md}{save-\/script}\mbox{]}


\item 

\mbox{[}!include\doxylink{vs-return-unity_8md}{return-\/unity}\mbox{]}


\item 

\mbox{[}!include\doxylink{vs-regen-node-library_8md}{regen-\/node-\/library}\mbox{]}


\end{DoxyEnumerate}

\begin{quote}
\mbox{[}!\+NOTE\mbox{]} If you change the namespace or namespaces used in your script, you must include the old namespace or namespaces to use the {\ttfamily \mbox{[}Renamed\+From\mbox{]}} attribute. \end{quote}
