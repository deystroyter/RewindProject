 Input nodes are an Event node type that can read input from \doxylink{namespace_unity}{Unity}\textquotesingle{}s \href{https://docs.unity3d.com/Manual/class-InputManager.html}{\texttt{ Input Manager}} or \href{https://docs.unity3d.com/Packages/com.unity.inputsystem@latest}{\texttt{ Input System package}} for use in a Script Graph.

For more information about how to read and capture input in Visual Scripting, see \doxysectlink{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-capture-player-input}{Capture user input in an application}{0}.\hypertarget{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-input-nodes_autotoc_md1894}{}\doxysection{\texorpdfstring{Input System package nodes}{Input System package nodes}}\label{md__library_2_package_cache_2com_8unity_8visualscripting_0d1_89_82_2_documentation_0i_2vs-input-nodes_autotoc_md1894}
