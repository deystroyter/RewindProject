◆ scope

virtual SettingsScope UnityEditor.SettingsManagement.FileSettingsRepository.scope
get

Sets the SettingsScope this repository applies to.

By default, this repository implementation is relevant to the Project scope, but any implementations that override this method can choose to store this serialized data at a user scope instead.

SettingsScope.Project, meaning that this setting applies to project settings (the default); or SettingsScope.User, meaning that this setting applies to user preferences.
